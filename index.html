Projects
Blog App
I created an authenticated blog app where users can only create and update posts if they are signed in. The client side of this project was created using react and redux. The server side was created using express and mongoDB. Check it out here!
MunchBuddy
For our final project in CS52, my team designed and implemented a react-native mobile app that connects strangers to grab meals together. Users can create an account and get matched with someone based off an inputted time and location. Once users are matched, they can view each other's profiles and message each other to coordinate a meet up.
Booking Site
My partner, Annie Ke, and I redesigned the Dartmouth library study space booking site for a better user experience. We used react and redux to create dynamic feedback for users. For a cleaner design, we made use of Google's Material-ui component library. Check it out here!
Rewriting the Code
I am part of an effort to create an easier application screening process for the Rewriting the Code organization for women in computer science (check them out here!). The goal is to take the application information and consolidate it into a cleaner and more readable format for reviewers.
